COMPETITIVE PROGRAMMING

ACA introduces the freshers to the vast world of Competitive Programming. Projects related to Data Structures and Algorithms are also offered.

Machine Learning

ACA offers introductory and intermediate projects related to Machine Learning. In the past, we have offered projects in different subdomains of ML such as Reinforcement Learning, Deep Learning.

Blockchains

ACA offers introductory projects related to Blockchains and their applications such as Cryptocurrency, and other operations demanding safe and secure transactions

WEB DEVELOPMENT

ACA offers projects related to Web Development where mentees are taught basic HTML, CSS, JS and several frameworks commonly used in developing a website.

CYBER SECURITY

ACA offers projects like InfoSec which are based on Cyber Security concepts and introduces the mentees to the vast domain of Cyber Security and its subdomains

PROGRAMMING LANGUAGE SPECIFICS

ACA offers projects related to specific Programming Languages such as Rust, Go, Julia, and others that find use in several non-trivial problem cases.
